## Overview

Fiverr Boss is a powerful Google Chrome Extension designed to automate the process of scraping Fiverr gig buyers and performing deep Open Source Intelligence (OSINT) searches on them. It features an advanced dashboard, real-time license verification powered by Supabase, and a secure device-locking mechanism.

## Features ✨

*   **Fiverr Gig Scraping:** Automatically extracts buyer usernames from Fiverr gig reviews.
*   **Deep OSINT Search:** Scrapes Google, Bing, and visual search engines (Google Lens, Bing Images) to uncover buyer social media profiles (LinkedIn, Facebook, Instagram, Twitter, Pinterest).
*   **Intelligent CAPTCHA Handling:** Detects search engine CAPTCHAs and automatically pauses the pipeline for manual solving.
*   **Interactive Dashboard:** View real-time metrics on gigs found, usernames extracted, and images discovered.
*   **CSV Import/Export:** Import your own lists of buyers for OSINT processing or export scraped data to CSV format.
*   **Premium UI:** A beautiful, responsive side panel interface with modern aesthetics, progress bars, and live activity logs.
*   **Robust Licensing System:**
    *   Powered by **Supabase**.
    *   **Device Locking:** Licenses bind to a specific device ID to prevent unauthorized sharing.
    *   **Real-time Syncing:** Instantly verifies expiration dates and lock status on startup and via background polling.

## Installation 🛠️

1.  Clone this repository: `git clone https://github.com/zonaedh/Fiverr-Scrapper.git`
2.  Open Google Chrome and navigate to `chrome://extensions/`
3.  Enable **"Developer mode"** in the top right corner.
4.  Click **"Load unpacked"** and select the extension directory.
5.  Click the extension icon in your browser toolbar to open the Fiverr Boss side panel.

## Usage 💡

1.  **Activate License:** Enter your assigned `XXXXX-XXXXX-XXXXX-XXXXX` license key to unlock the extension.
2.  **Scrape Buyers:** Navigate to the **Scraper** tab, paste a Fiverr URL, and click **Start Scraping**.
3.  **Run OSINT:** Once buyers are found (or uploaded via CSV in the **Socials** tab), start the OSINT search to find their social media profiles.
4.  **Export:** Go to the **Dashboard** to export all compiled data into a clean CSV file.
